# Brand System

`DESIGN.md` is the canonical source of truth for the RMR visual system. This file is the shorter project-doc version for agents that need the current tokens and component rules quickly.

## Current Direction

Roll My Retirement is a warm editorial fintech site: calm, precise, image-led, and lightly asymmetric. It should feel trustworthy without becoming a generic financial services template.

- Atmosphere: gallery-airy, warm, confident.
- Theme: light cream surfaces with one charcoal accent.
- Type: Instrument Serif for display, Inter for UI/body.
- Motion: GSAP/ScrollTrigger/Lenis with restrained entrances and scroll-linked details.

## Typography

- Display: `Instrument Serif`, 400, tight tracking.
- Body/UI: `Inter`, 300-600.
- Eyebrows: 11px, uppercase, 0.12em tracking, muted color.
- Body copy: 15-16px with 1.65-1.7 line-height.

## Components

- Buttons are pill-shaped; primary uses `--bg-dark`, secondary uses transparent fill and `--border-strong`.
- Cards use `--bg-card`, `--border`, `--radius-card`, and very soft hover elevation.
- Forms use labels above inputs, inline errors, and radio selector pills for choice groups.
- Blog images are editorial assets with explicit object-position where crop matters.
- Footer is generated by `website/scripts/footer.js`; all pages mount `<div id="site-footer"></div>`.

## Footer And Dark Sections

Default footer and footer CTA sections are light (`--bg`). The article detail page currently retains a dark CTA as an intentional contrast moment. Advisor process sections can use `--bg-dark`; do not treat dark footers as the default system.

## Assets

- Current public social share image: `Website Images/rmr-social-share-2.png`.
- Source images live in `website/assets/images/`; the build preserves the public `Website Images/` output path.
- Source favicon/logo files live in `website/assets/icons/`; the build emits public logo and favicon files to the `dist/` root. `rmr-logo-new.png` is the primary nav/footer mark; `rmr-logo.png` remains the original design-system variant.
- Non-published source/back-pocket assets live under `docs/source-assets/`.
