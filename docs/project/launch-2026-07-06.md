# Production Launch Handoff - July 6, 2026

_Context for future agents and humans after the Roll My Retirement production cutover from Framer to Netlify._

## Summary

On July 6, 2026, the RMR website was cut over from the old Framer-hosted public site to the new static Netlify site. The production domain is now served from Netlify, canonical URLs use `https://www.rollmyretirement.com`, and Google Search Console has a fresh sitemap showing 12 discovered pages.

## Current Production State

- Primary production host: `https://www.rollmyretirement.com`
- Apex redirect: `https://rollmyretirement.com` redirects to `https://www.rollmyretirement.com/`
- Netlify app domain: `https://rollmyretirement.netlify.app`
- Netlify project: `rollmyretirement`
- GitHub repo: `https://github.com/wwwhatevertyler/rmr-website`
- Build command: `npm run build`
- Publish directory: `dist`
- Netlify config: `netlify.toml`

Important note: during the cutover, macOS/browser DNS cache briefly kept resolving `www.rollmyretirement.com` to the old Framer IPs even after GoDaddy, Cloudflare DNS, Google DNS, and Netlify were correct. If this reappears, confirm with authoritative/public DNS before changing records.

## Netlify Domain State

Netlify custom domains were configured as:

- `www.rollmyretirement.com` - primary domain
- `rollmyretirement.com` - redirects automatically to primary domain
- `rollmyretirement.netlify.app` - Netlify app subdomain

- page-specific `<title>`
- page-specific meta description
- canonical URL
- Open Graph title/description/url/image
- Twitter title/description/image
- Article JSON-LD
- embedded post data
- visible article body HTML in the first response

## If Something Looks Wrong Later

Start here:

1. Check Netlify deploy status for the latest `main` commit.
2. Confirm `npm run build` succeeds locally.
3. Check DNS at the source:
   - `dig @ns57.domaincontrol.com +short www.rollmyretirement.com CNAME`
   - `dig @ns57.domaincontrol.com +short rollmyretirement.com A`
4. Compare public DNS:
   - `dig @1.1.1.1 +short www.rollmyretirement.com CNAME`
   - `dig @8.8.8.8 +short www.rollmyretirement.com CNAME`
5. Check the custom domain through Netlify edge:
   - `curl -I https://www.rollmyretirement.com/`
   - `curl -I https://rollmyretirement.com/`
   - `curl -I https://www.rollmyretirement.com/sitemap.xml`

Avoid changing GoDaddy records unless authoritative DNS no longer matches the intended DNS table above.
